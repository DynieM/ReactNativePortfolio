To Do App


Here is a To Do App. I created this with React. The app allows users to add tasks to a list and check them off when completed. I had an issue with checking off the items so I found documentation that helped guide me though it. The items are listed in the center and the input to enter more tasks is at the bottom of the page.

Design



Flash Card App Design


This is a study card app designed using Figma. It allows the user to add questions and their corresponding answer to a set of study cards, edit and delete a flash card, tap to reveal the question's answer, and view the cards in random order. The hardest part of this was maximizing my screen space without overloading it with too much content. The design is black text with a purple background.

To Do App Design


I designed a To Do app that allows the user to sign in and out, add a new task with a description and deadline, sort the task list by completion status, mark tasks as complete, and delete tasks. The hardest part about this was making sure that all of the features were available and made sense placement wise. The design is white text and a sky blue background.
